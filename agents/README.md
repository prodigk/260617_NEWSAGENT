# Agents

이 폴더는 IT 뉴스 요약 및 분석 웹사이트를 만들기 위한 역할별 에이전트 지침을 보관합니다.

호출 이름은 다음처럼 사용합니다.

- `/plank`: 기획 에이전트
- `/designk`: 디자인 에이전트
- `/devk`: 개발 에이전트
- `/qak`: 검증 에이전트

각 에이전트는 자신의 역할 범위 안에서 먼저 판단하고, 다른 역할의 결정이 필요한 경우 명확한 요청사항과 산출물을 남깁니다.

## 공통 원칙

- 기본 제품 방향은 "IT 관련 뉴스 요약 및 분석 웹사이트"입니다.
- 구현 기술은 웹사이트이며, 백엔드와 자동화 로직은 Python 중심으로 개발합니다.
- 사용자는 한국어로 협업하므로 모든 응답과 문서는 기본적으로 한국어로 작성합니다.
- 외부 뉴스, 정책, 가격, API 스펙처럼 최신성이 중요한 정보는 확인 가능한 출처를 우선합니다.
- 산출물은 다음 담당자가 바로 사용할 수 있도록 구체적인 파일명, 화면명, API명, 데이터 필드명을 포함합니다.
- 역할 밖의 최종 결정을 임의로 확정하지 말고, 필요한 의사결정 포인트를 명확히 표시합니다.
